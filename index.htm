GIT Branch

Đây là cách chia nhiều nhóm làm việc trên cùng 1 dự án trên github

1. Xử lý branch trên máy máy tính

  • Xem nhánh (branch): git branch -a
  • Tạo nhánh (branch): git branch -c tên_nhánh (nó sẽ lấy nguyên code trên branch đang đứng)
  • Chuyển đổi giữa các branch: git checkout tên_nhánh
  • Tạo nhánh đồng thời chuyển sang nhánh mới tạo: git checkout -b tên_nhánh
  • Xóa nhánh: git branch -d tên_nhánh, buộc xóa thay -d = -D

2. Sử lý branch trên github

  • Upload branch lên github (ko đc xóa nhánh đang đứng): git push -u origin tên_nhánh
  • Xóa nhánh: git push -d origin tên_nhánh
